# 📂 Universal File Manager (Python)

A simple command-line based Universal File Manager built using Python and pathlib module.  
This project supports full CRUD operations (Create, Read, Update, Delete) for all file types.

## 🚀 Features

- 📄 List all files (recursive search)
- ➕ Create new files (auto folder creation supported)
- 📖 Read file content
- ✏️ Update files (Rename / Overwrite / Append)
- 🗑️ Delete files (with confirmation)
- 🔁 Continuous menu-driven system
- 🛡️ Exception handling implemented

## 🛠️ Technologies Used

- Python 3.x
- pathlib module
- Exception Handling
- File Handling


## ▶️ How to Run

1. Make sure Python is installed.
2. Open terminal in project folder.
3. Run the following command: python file_manager.py
